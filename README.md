### **Requirements**
1. Kubernetes cluster ready
2. Kubectl installed 
### **How to**
1. Clone repository
```
https://github.com/ludesdeveloper/kubernetes-knative-kourier.git
```
2. Change directory
```
cd kubernetes-knative-kourier
```
3. Install with script
```
./install_knative.sh
```
